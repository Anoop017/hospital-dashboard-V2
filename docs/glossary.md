Work Flows
<=========>


Patient
<------>
Patient Logins --> Search for Doctors/Search for Doctors Types(like childrens doctors, skin doctor, dentist etc.) --> View available timings --> Book appointments.

Patients Logins --> See the dashboard --> See the health status/reports --> can see the medical bills etc

Doctor
<----->
Doctor Logins --> View/Update the appointmnets --> allow or reject the appointments

Doctor Logins --> update the report of patients health condition etc


Admin
<---->
Admin Logins --> Can see all the registered users (both doctor and patients) --> update or delete or create new users.

Admin Logins --> see an update reports/bills
